# Hangman Game - React

A modern implementation of the classic Hangman game built with React , featuring smooth gameplay and a responsive design.

## Features

- 🎮 Classic Hangman gameplay with keyboard and on-screen inputs
- 📝 Curated word list with easy customization
- 🎨 Clean, responsive UI with visual hangman progression
- ⌨️ Keyboard event support for quick gameplay

## Installation & Running

1. Clone the repository:
   ```bash
   git clone https://github.com/mahin273/HangMan.git
   ```

2. Navigate to the project:
   ```bash
   cd react-hangman
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173) to play!

For production build:
```bash
npm run build
```

## How to Play

- Guess letters to reveal the hidden word
- Each incorrect guess adds a part to the hangman
- Guess all letters correctly to win!
